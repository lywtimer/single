  • 文本属性-字体大小

    • text-size: 40px;
    • 由于字体设计原因,文字最终呈现的大小,并不一定与font-size的值一致,可能大,也可能小
    • 通常情况下,文字相对字体设计框,并不是垂直居中,通常都靠下一些.

  • 文本属性-行高

    • line-height: 60px; line-height: 1.6; line-height: 160%;
    • 可选值:
      1. normal: 由浏览器根据文字大小决定的一个默认值
      2. 像素 px
      3. 数字: 参考自身 font-size 的倍数
      4. 百分比: 参考自身 font-size 的百分比
    • 注意事项:
      1. line-height过小会产生文字重叠,最小值是0
      2. line-height可以继承,为了更好的呈现文字,最好写数值
      3. line-height 和 height 的关系 设置了height,那么高度就是height的值 不设置height,就会根据line-height计算高度
      4. 行高的应用场景:
        1. 调整多行文字的间距
        2. 调整单行文字的垂直居中
    • 由于字体设计原因,文字在一行中,并不是绝对居中,行高设置不当会导致行与行之间重叠,效果就是文字会重叠.

  • 文本属性-垂直对齐

    • 顶部:无需任何属性
    • 居中:单行文字使用 height=line-height
    • 底部:单行文字使用 line-height=height*2 - font-size - x; x 根据字体族进行调整

  • 文本属性-vertical-align

    • 作用:指定同一行元素之间,或表格单元格内文字的垂直对齐方式
    • 常用值:
      1. baseline 默认值
      2. top 元素顶部与所在行顶部对齐
      3. middle 使元素的中部与父元素的基线加上父元素 x-height（译注：x 高度）的一半对齐。
      4. middle 使单元格内边距盒模型在该行内居中对齐。
      5. bottom 元素底部与所在行底部对齐
