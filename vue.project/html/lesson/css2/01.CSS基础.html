css 全称 层叠样式表

层叠:一层一层涂上去

表:样式属性的列表

一种标记语言

核心思想:HTML 搭建结构, CSS 添加样式, 结构与样式的分离

CSS的编写位置

  • 行内样式:style="名:值;"
  • 内部样式:写在html的内部,将CSS代码提取至style标签里,可以在当前页面复用了
  • 外部样式:写在.css文件中;可以在不同页面复用了;结构与样式的完全分离;触发缓存机制
  • 样式表的优先级:

    行内样式优先级高;内部与外部样式,后来者居上(覆盖效果)

语法规范

选择器:找到元素

声明快:多个声明组成;声明的格式 属性名: 属性值

CSS代码风格

展开风格,紧凑风格

项目上线时使用webpack把展开风格压缩成紧凑风格,减小文件的体积

基本选择器

通配选择器 元素选择器 类选择器 id选择器
