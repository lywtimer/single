长度单位

  • 第一种长度单位:px 像素
  • 第二种长度单位:em 当前元素或父元素的font-size的倍数看看我多大
  • 第三种长度单位:rem 相对于根元素的font-size的倍数看看我多大
  • 第四种长度单位:% 相对于父元素的百分比看看我多大

元素的显示模式

  • 块元素 block
    • 在页面中独占一行,从上往下排列
    • 默认宽度:撑满父元素
    • 默认高度:由内容撑开
    • 可以通过css设置宽高
    • 常用标签 html body h1 h6 hr p pre div ul ol li dl dt dd table tbody thead tfoot tr caption form option
  • 行内元素 inline
    • 在页面中不独占一行,一行不能容纳的行内元素,会在下一行继续从左到右排列
    • 默认宽度:由内容撑开
    • 默认高度:由内容撑开
    • 无法通过css设置宽高
    • 常用标签 br em strong sup sub del ins a label
  • 行内块元素 inline-block
    • 在页面中不独占一行
    • 默认宽度:由内容撑开
    • 默认高度:由内容撑开
    • 可以通过css设置宽高
    • 常用标签 img td th input textarea select button iframe

盒子模型的组成部分

所有的元素都是盒子模型

从里到外依次是 content padding border margin

  • margin 盒子与外界的距离
  • border 盒子的边框
  • padding 紧贴内容的补白区域
  • content 元素中的文本或者后台元素都是它的内容

盒子内容区

  • width 内容区域宽度
  • max-width 内容区域最大宽度
  • min-width 内容区域最小宽度
  • height 内容区域高度
  • max-height 内容区域最大高度
  • min-height 内容区域最小高度

注意:

max-width、min-width 一般不与 width 一起使用

max-height、min-height 一般不与 height 一起使用

默认宽度

  • 不设置 width 属性时, 元素呈现出来的宽度
  • 总宽度 = 父 content - 自身的左右 margin
  • 内容区的宽度 = 父 content - 自身的左右 margin - 自身的左右 border - 自身的左右 padding

内边距

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
  • padding: 10px; 四个方向内边距都是 10px
  • padding: 10px 20px; 上下 10px, 左右 20px
  • padding: 10px 20px 30px; 上 10px, 左右 20px, 下 30px
  • padding: 10px 20px 30px 40px; 上 10px, 右 20px, 下 30px, 左 40px (顺时针)

注意点:

padding 不能为负数

行内元素的左右内边距没有问题.上下内边距可能导致遮挡的问题

块级元素、行内块元素没有问题.

  • border-*-?
  • * = top,bottom,left,right

外边距

  • 属性样式与padding一致
  • 子元素的margin是参考父元素的content计算的
  • 上左 margin 影响自身的位置, 下右 margin 影响兄弟的位置
  • 行内元素,左右的margin可以完美设置.上下的margin设置无效
  • 对于块级元素,margin的值可以是auto,左右margin设置auto可以实现在父元素内水平居中
  • margin的值可以是负数
  • margin的塌陷问题结局方案
    1. 给父元素设置不为0的padding
    2. 给父元素设置宽度不为0的border
    3. overflow: hidden;
  • 上下铺兄弟的margin合并问题
    1. 上面兄弟元素的下外边距和下面兄弟的上外边距会合并,取一个最大值,而不是相加
    2. 无需解决,只给一个设置上下外边距就可以了

处理内容溢出

  • overflow 溢出内容的处理方式
    1. visible 显示,默认值
    2. hidden 隐藏
    3. scroll 显示滚动条,不论内容是否溢出
    4. auto 自动显示滚动条
  • 不建议使用overflow-x overflow-y

隐藏元素的两种方式

  • visibility属性,默认show,可以设置为hidden,元素虽然隐藏,但是还占有原来的位置
  • display:none; 彻底隐藏元素,不占用任何位置

样式的继承

  • 会继承的css属性 字体属性、文本属性(除了vertical-align)、文字颜色等.
  • 不会继承的css属性 边框、背景、内边距、外边距、宽高、溢出方式等.
  • 总结:能继承的属性,都是不影响布局的.
