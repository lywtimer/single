颜色

  • 颜色名 编写简单,名称有限,不够精细 颜色名参考
  • RGB 光的三原色 red green blue
    • rgba(300, 200, 100, 0.7) RGBA 取值0~255 或者 100%, 不能混合; A 表示透明度,取值范围 0~1
    • 三种颜色值相同,呈现灰色,值越大,灰色越浅
  • 十六进制色值 HEX或者HEXA 原理与 rgb 一致 ,书写格式:#c7edcc
    • 色值范围 00~ff
    • 色值的两位都相同可以简写: #ff9988 简写 #f98
    • 色值的两位都相同可以简写: #ff998866 简写 #f986
    • ie浏览器不支持HEXA
  • HSL 或者 HSLA ,书写格式 hsla(0deg, 100%, 70%, 1)
    • 通过 色相,饱和度,亮度, 来表示一个颜色
    • 色相取值 0~360
    • 饱和度取值 0~100% 0%全灰 100%没有灰
    • 亮度取值 0~100% 0%黑色 100%白色
