  • 字体属性-字体族

    培养以练习为导向的心态,即不再认为工作日只能用来工作,练习只能在特殊时刻,特定场合才能进行,而是将日常商业活动转变成练习活动,将练习变为日常工作的一部分.
    • font-family: "SimSun","楷体"; 使用双引号包裹,按顺序匹配,不合规则或者不存在往后查找,找到就用
    • 使用 serif 或 sans-serif 兜底
    • 使用英文名兼容性更好,可以在网上查询或者电脑设置中查看
    • (非)衬线字体(非衬线字体使用频繁)

  • 字体属性-字体风格

    • font-style: italic;
    • 常用值: normal:正常(默认) italic:斜体(使用字体自带的斜体效果) oblique:斜体(强制倾斜产生的斜体效果)
    • 实现斜体效果时推荐使用 italic

  • 字体属性-字体粗细

    • font-weight: lighter;
    • 常用值: lighter:细 normal:正常(默认) bold:粗 bolder:很粗(多数字体不支持,会使用bold) 100~1000 且无单位,数值越大,字体越粗 100~300等同于lighter,400~500等同于normal,600以上等同于bold.与字体设计精度有关
    • 实现斜体效果时推荐使用 italic

  • 字体属性-复合写法

    • font: italic lighter 50px "楷体";
    • 编写规则
      1. 字体大小、字体族必须都写上
      2. 字体大小、字体族顺序固定倒数第二、第一位
      3. 各个属性间使用空格隔开
    • 开发中更推荐复合写法
