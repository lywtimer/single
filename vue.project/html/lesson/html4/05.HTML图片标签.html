图片类型

  • jpg 有损压缩,支持的颜色丰富,占用空间较小,不支持透明背景,不支持动图;
  • png 无损压缩,支持的颜色丰富,占用空间略大,支持透明背景,不支持动图;
  • bmp 不压缩,支持的颜色丰富,占用空间极大,不支持透明背景,不支持动图;
  • gif 支持的颜色少,支持简单透明背景,支持动图;
  • webp 具备以上优点,兼容性不太好
  • base64 把图片进行base64编码,需要通过浏览器打开,一些较小的图片或者需要和网页一起加载的图片
